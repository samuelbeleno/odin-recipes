# Odin recipes
The purpose of this project is to create the HTML code for a website featuring recipes for our favorite dishes.

The website will briefly describe each dish, detail the required ingredients, and provide step-by-step instructions for preparing it.

The ultimate goal of this project is to demonstrate mastery of basic HTML (links, image insertion, links to other files, text, headings, and lists).
